# Todo App

A simple command-line Todo application built in Rust. This app allows users to manage their tasks easily from the terminal.

## Features
- Add new tasks
- List all tasks
- Mark tasks as completed
- Persist tasks in a JSON file for data storage

## Technologies Used
- Rust
- serde for serialization
- serde_json for JSON handling
